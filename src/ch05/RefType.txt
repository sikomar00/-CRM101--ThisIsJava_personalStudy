데이터 타입 분류
    기본
    참조

객체 = 데이터 + 메소드

기본 타입 변수와 참조 타입 변수의 차이: 저장되는 값
    기본: 값 자체 저장
    참조: 객체가 생성된 메모리 번지 저장 ~ 참조타입 String 타입 변수는 번지를 통해 String 객체를 참조

메모리: 메소드, 힙, 스택
    메소드: 바이트코드 파일을 읽는 내용이 저장
        바이트코드: .class
    힙: 객체가 생성되는 영역. 객체의 번지는 메소드 영역과 스택 영역의 상수와 변수에서 참조
    스택: 메소드를 호출할때마다 생성되는 프레임이 저장되는 영역
